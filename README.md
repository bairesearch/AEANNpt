### Installation
```
conda create -n pytorchsenv
source activate pytorchsenv
conda install python
pip install datasets
pip install torch
pip install lovely-tensors
pip install torchmetrics
```

### Execution
```
source activate pytorchsenv
python ANNpt_main.py
```
